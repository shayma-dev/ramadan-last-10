# 🌙 Last 10 Nights of Ramadan: Maximum Ajr Plan

A structured, nightly worship plan for the last 10 nights of Ramadan built entirely on **authentic (Sahih) hadith and Quran**. No weak or fabricated narrations.

## What This Is

A single-page interactive guide covering:
- A nightly schedule from **9:00 PM → Fajr (4:30 AM)**
- A full **Quran Khatm** plan (3 juzz per night over 10 nights)
- **~1,000 ayat in prayer** every night, reaching the level of *Al-Muqantareen*
- All dhikr targets with running counts distributed across the night
- Two alternating prayer types (Type A: Juzz Tabarak + Amma / Type B: The 12 Surahs)
- Every hadith shown in **full Arabic and English** with source reference
